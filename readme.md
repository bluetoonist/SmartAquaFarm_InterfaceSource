# README

# Installation Process(설치 안내)
- JAVA - JDK 8 이상  
- apache tomcat 8.5 이상  
- Data Base - Oracle 11g 

*위의 환경 구성 후 Eclipse를 통한 Dynamic Web Project에서 작동 가능
